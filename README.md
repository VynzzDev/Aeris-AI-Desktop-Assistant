## Getting Started

You need to install Python on your machine, i recommend using Python 3.12.9

### Prerequisites

* [Python](https://www.python.org/downloads/release/python-3129/)
* OpenRouter API Key [Get it here](https://openrouter.ai/settings/keys)
* SerpAPI API Key [Get it here](https://serpapi.com/dashboard)

### Installation

1. Clone the repo/Download the repo
   ```sh
   git clone https://github.com/VynzzDev/Aeris-AI-Desktop-Assistant.git
   ```
2. Run aeris.py
   ```sh
   python aeris.py
   ```
3. That's it, all dependencies will automatically download, and you will be asked to enter api key.

## Features

- [ ] Open APP
    - AERIS can open any apps in your computer based on your command. 
- [ ] Message Sender
    - AERIS can send message to your contact via Whatsapp, Telegram, etc.
      For Example :
         - You say "send message to my dad, say Hi." Aeris will automatically detect chat app in your computer and open it then search for the contact name and send       message to them.
- [ ] Web Searching
    - AERIS can help you search for information on the internet in real time and open the browser automatically and search for topics according to your commands.
- [ ] Weather report
    - AERIS can search realtime weather data in your current location
- [ ] Chat Mode
- [ ] Live Talk Mode
- [ ] Aircraft Report
    - This is just an entertainment feature, this feature can detect aircraft around the LAT & LON that you set in the `aircraft_report.py` file in real time.

See the [open issues](https://github.com/VynzzDev/Aeris-AI-Desktop-Assistant/issues) for a full list of proposed features (and known issues).
